# SANGFOR Partner Workflows

## 1. Sales Workflow

```text
Lead Intake
 → Customer Creation
 → Opportunity Creation
 → Qualification
 → Sales Manager Review
 → Discovery Assignment
```

산출물:

- Lead Brief
- Qualification Report
- Customer Profile

승인:

- Opportunity Gate

## 2. Presales Workflow

```text
Discovery
 → Product-family Checklist
 → Solution Fit Matrix
 → Sizing
 → Architecture Review
 → BoM Draft
```

산출물:

- Discovery Note
- Sizing Worksheet
- Solution Fit Matrix
- BoM Draft

승인:

- Solution Fit Gate

## 3. Commercial Workflow

```text
BoM
 → Product Line Items
 → Service Line Items
 → Discount Request
 → Margin Simulation
 → Commercial Approval
 → Quote Version
```

산출물:

- Quote Draft
- Margin Report
- Discount Request

승인:

- Commercial Gate

## 4. Vendor Request Workflow

```text
Need Identified
 → Vendor Request Create
 → Vendor Liaison Review
 → Submit via Partner Portal or Manual Channel
 → Vendor Response Record
 → Apply to Quote/PoC/Support
```

요청 유형:

- special_discount
- demo_license
- nfr_asset
- technical_escalation
- product_clarification
- certification_support

## 5. PoC Workflow

```text
PoC Scope
 → Success Criteria
 → Resource Reservation
 → Demo License Request
 → Customer Test Scenario
 → Execution
 → Result Review
 → Conversion Decision
```

필수 항목:

- success criteria
- customer sponsor
- demo license/equipment
- rollback plan
- exit criteria

## 6. Delivery Workflow

```text
SOW
 → Delivery Readiness
 → Engineer Assignment
 → Deployment
 → Configuration Backup
 → License Activation
 → Acceptance Test
 → Handover
 → Support Entitlement Start
```

납품 완료 시 자동 생성:

- Customer Asset
- License Record
- Subscription Record
- Renewal Schedule

## 7. Support Workflow

```text
Ticket Intake
 → Severity Classification
 → SLA Timer Start
 → Troubleshooting
 → Vendor Escalation if needed
 → Resolution
 → RCA
 → Customer Confirmation
```

## 8. Renewal Workflow

```text
T-90 days Renewal Opportunity
 → Account Manager Follow-up
 → Health Check
 → Renewal Quote
 → Commercial Approval
 → Customer Proposal
 → Renewal Closed
```

## 9. Upsell Workflow

```text
Asset/Support History Analysis
 → Upsell Candidate
 → Account Review
 → Discovery Lite
 → Proposal
```

예시:

- NGFW 고객 + ransomware concern → EPP/XDR/MDR
- HCI 고객 + DR 요구 → backup/DR/managed cloud
- VDI 고객 + remote work 확대 → VDI expansion
