# SANGFOR Partner Business Fit Review

## 판정

V2의 기본 흐름은 맞지만, Platinum Partner 운영 시스템이 되려면 다음을 반드시 강화해야 한다.

1. Deal Qualification Engine
2. Product SKU / License Catalog
3. Vendor Request Workflow
4. Quote Cost & Margin Engine
5. PoC Resource Management
6. Customer Asset / License Lifecycle
7. Renewal Automation
8. Support SLA / Vendor Escalation
9. Engineer Certification Matrix
10. Sales Forecast Dashboard

## SANGFOR Partner 업무 특성

SANGFOR 공식 자료 기준으로 파트너 프로그램은 Certified, Gold, Platinum tier를 제공하며, Partner ONE 포털은 채널 파트너가 효율적으로 운영할 수 있도록 자료와 workflow를 제공한다. 이 시스템은 Partner ONE을 대체하지 않고, 회사 내부의 고객/딜/자산/승인/갱신 운영 시스템으로 동작해야 한다.

## 핵심 업무 흐름

```text
Lead
 → Qualification
 → Vendor/Deal Check
 → Discovery
 → Product-family Sizing
 → Solution Fit
 → BoM
 → Vendor Discount/Demo License
 → Quote
 → Commercial Approval
 → Proposal
 → PoC
 → Delivery
 → Asset/License Activation
 → Support
 → Renewal
 → Upsell
```

## Business Fit Gap

| 영역 | 현재 | 보완 |
|---|---|---|
| Lead | 고객 등록 가능 | lead source, qualification score |
| Opportunity | 기본 딜 | 예상 매출, 경쟁사, close probability |
| Discovery | 문서 중심 | 제품군별 checklist |
| Product | 제품명 수준 | family/product/SKU/license/sizing |
| Quote | line item | vendor discount, service cost, promotion |
| PoC | plan/result | demo license, equipment, success criteria |
| Delivery | checklist | license activation, asset creation |
| Support | ticket/RCA | SLA, vendor escalation, affected asset |
| Renewal | report | 만료일 자동 opportunity |
| Partner Operations | persona만 존재 | vendor request object 필요 |

## Deal Qualification Score

```text
score = Budget + Authority + Need + Timeline + Technical Fit + Strategic Fit - Competitive Risk
```

| 항목 | 질문 |
|---|---|
| Budget | 예산이 확인되었는가 |
| Authority | 의사결정권자가 확인되었는가 |
| Need | 고객 pain point가 명확한가 |
| Timeline | 도입 일정이 현실적인가 |
| Technical Fit | SANGFOR 제품군과 맞는가 |
| Strategic Fit | 레퍼런스/확장 가능성이 있는가 |
| Competitive Risk | 경쟁사 lock-in이 강한가 |

## Platinum Partner 운영 관점

Platinum Partner급 회사는 단순히 문의를 처리하는 것이 아니라 다음을 관리해야 한다.

- 제품군별 pipeline
- 프리세일즈 리소스
- 벤더 특별가/데모 라이선스
- 엔지니어 인증
- 고객 자산
- 갱신 매출
- SLA와 escalation
- 제품군별 수익성
