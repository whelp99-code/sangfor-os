# 13_COLOR_AGENT_ORG

이 폴더는 V3.2에서 추가된 Hermes Color Agent Organization 문서 모음이다.

## 목적

Agentic Company OS / SANGFOR Partner OS에 색상 기반 Agent 협업 구조를 추가한다.

## 포함 문서

| 파일 | 설명 |
|---|---|
| Color_Role_Definitions.md | Blue/Red/Orange/Gray/Teal 역할 정의 |
| Kanban_Handoff_Template.md | Agent 간 책임 이관 카드 템플릿 |
| Color_Agent_Routing_Rules.md | 위험도 기반 Color Agent 참여 규칙 |
| SANGFOR_Color_Mapping.md | SANGFOR Partner OS에 색상 역할을 매핑한 문서 |
| Color_Agent_Adoption_Guide.md | 기존 V3.1 시스템에 V3.2를 도입하는 절차 |

## 핵심 원칙

```text
Color Agent는 직무 Persona가 아니다.
Color Agent는 검토 관점, 책임 경계, handoff 표준, 품질 게이트다.
```
