# Sangfor MCP Workflow — 프로젝트 블루프린트

**작성일**: 2026-06-10  
**버전**: v2.0  
**목적**: Sangfor 전문 → 멀티 벤더 보안 솔루션 컨설팅 플랫폼으로 진화

---

## 1. 프로젝트 비전

### 현재 위치
```
Sangfor 제품 전문 MCP 서버
  ↓
감사항목 → Sangfor 제품 매핑 → 설정 가이드 생성
```

### 목표 위치
```
멀티 벤더 보안 솔루션 컨설팅 플랫폼
  ↓
감사항목 → 벤더별 솔루션 비교 → 최적 대안 제안 → 프로젝트 획득
```

### 핵심 가치 제안
| 구분 | Sangfor 전문가 | 멀티 벤더 컨설턴트 |
|------|---------------|-------------------|
| **고객 인식** | "Sangfor 파는 사람" | "보안 전문가" |
| **프로젝트 범위** | Sangfor만 | 전체 보안 솔루션 |
| **경쟁력** | 제품 지식 | 솔루션 비교 + 최적 추천 |
| **고객 신뢰도** | 중간 | 높음 |

### 전체 구조


### 데이터 흐름
```
1. 입력: ITAC Excel 체크리스트
   ↓
2. 파싱: Result 있는 항목만 추출 (18개)
   ↓
3. 분석: 카테고리별 솔루션 매핑
   ↓
4. 비교: 벤더별 솔루션 장단점 분석
   ↓
5. 추천: 최적 솔루션 조합 제안
   ↓
6. 출력: 설정 가이드 + 비교 보고서
```
